# Бот для посещаемости сотрудников (Google Sheets + Хостинг)

Этот бот предназначен для учета посещаемости сотрудников. Данные сохраняются в Google Таблицу, что позволяет бесплатно размещать бота в интернете (на Render, Koyeb и др.) без потери данных.

## Возможности
- Регистрация сотрудников (ФИО).
- Отметка "Пришел" (ставит + в таблицу).
- Отметка "Ушел раньше" (спрашивает количество часов и записывает их).
- Сводная таблица (Summary) с датами.
- Доступ к таблице по ссылке для админа.

## Настройка Google Sheets

1.  Создайте проект в [Google Cloud Console](https://console.cloud.google.com/).
2.  Включите **Google Sheets API** и **Google Drive API**.
3.  Создайте **Service Account**, скачайте JSON-ключ.
4.  Переименуйте ключ в `credentials.json` и положите в папку с ботом.
5.  Создайте пустую Google Таблицу.
6.  Добавьте email вашего Service Account (из JSON-файла) в список редакторов таблицы.
7.  Скопируйте ID таблицы из URL и вставьте в `.env` в поле `GOOGLE_SHEET_ID`.

## Размещение в интернете (Render.com)

1.  Создайте аккаунт на [Render.com](https://render.com/).
2.  Создайте новый **Web Service** или **Background Worker**.
3.  Подключите ваш GitHub репозиторий с кодом бота.
4.  В настройках Render (Environment Variables) добавьте все переменные из `.env`.
5.  В поле `Start Command` укажите: `python bot.py`.

## Локальный запуск
1. Установите зависимости: `pip install -r requirements.txt`.
2. Запустите: `python bot.py` (или через `start.bat`).
